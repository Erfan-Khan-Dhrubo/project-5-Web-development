Question-1: What are the different ways to select an element in the DOM?

There are different ways to select an element in the DOM, such as:
getElementById: Selects an element by its unique ID attribute.
getElementsByClassName: Selects all elements with a specified class name, returning an HTMLCollection.
getElementsByTagName: Selects all elements with a specified tag name, also returning an HTMLCollection.
querySelector: Selects the first element that matches a specified CSS selector.
querySelectorAll: Selects all elements that match a specified CSS selector, returning a NodeList.

Question-2: What is the difference between innerHTML, innerText, and textContent ?

The differences between innerHTML, innerText, and textContent are shown below :-
innerHTML: Deals with HTML markup and tags, allowing you to retrieve or set the HTML content within an element.
innerText: Handles the visible text content of an element, ignoring HTML tags and considering CSS styles.
textContent: Provides the plain text content of an element, including all text from child elements without HTML tags and ignoring CSS styles.

Question-3: What is event delegation in the DOM?

Event delegation is a way to handle events for multiple child elements using a single event listener on the parent element. When an event happens on a child, it bubbles up to the parent, which can then handle the event. This makes your code more efficient and easier to manage, especially when adding or removing elements.

Question-4: What is event bubbling in the DOM?

Event bubbling is when an event starts at the target element and moves up to its parent elements. For example, if you click a button inside a div, the click event first triggers on the button and then moves up to the div and beyond. This allows parent elements to handle events from their children.

Question-5: How do you create, add, and remove elements using JavaScript?

We can create, add and remove elements using JavaScript in below way:-
Create: document.createElement to create a new element.
Add: Use appendChild or insertBefore to add the element to the DOM.
Remove: Use removeChild or remove to remove the element from the DOM.
